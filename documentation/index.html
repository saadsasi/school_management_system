ProSchool

ProSchool Management System in PHP (Laravel) - Online ProSchool Management System

  • Author: vipuldayani
  • Author Portfolio: https://codecanyon.net/user/vipuldayani

Introduction

ProSchool is a script built with laravel php framework. You can manage a complete event or conference using this php script. A ProSchool management system is an information management system for educational institutions to manage Admin, Teacher, Student and Parent data.

This CMS can easily be modified by most of the developers due to clean and easy coding system. It is strong against web vulnerabilities; and your website will be safe. Also in all pages, admin level security and invalid page url submission checking were done.

Features

This CMS script has the following features:

  • Easy and simple interface to use
  • Fully responsive for any kind of device
  • Modern Design with Bootstrap 4
  • Clean and understandable coding
  • Secured coding against web vulnerabilities
  • Direct access or invalid URL press stopped for each pages
  • RTL Layout supports
  • Payment setting
  • PayPal and Stripe Integration
  • Dynamic of data in dashboard
  • Sending emails to subscribers from back end
  • Unlimited photo upload and management
  • Admin panel login photo manage
  • Admin panel forget password option
  • Title and Meta Description setup for SEO of each pages
  • SEO friendly URL structure

Requirements

Following modules must be activated on your server:

  1. PHP version: 8.0 or higher
  2. PDO Extension
  3. GD Library
  4. cURL
  5. allow_url_fopen
  6. Mbstring

Installation

In order to install the CMS, please follow these steps:

  1. Download the main zip file from Codecanyon and extract it. You will get total 3 folders in there: database, documentation and cms.
  2. Compress the 'cms' folder and give name something like cms.zip. Then upload this file into your server using FTP or Cpanel. Extract this and use. You can use our php script into root domain or subdomain.
  3. Create mysql database using cpanel or your hosting provider's system. Then import the proschool.sql file (you will get it in database folder) into your created database.
  4. Now you will have to edit the configuration file. Location of the configuration file is here: root > .env .

  5. Open up the .env file with any text editor (for example: Notepad++) and change the text in there as following:

  6. Now you will have to edit the configuration file. Location of the configuration file is here: root > config > database.php.

  7. Open up the database.php file with any text editor (for example: Notepad++) and change the text in there as following:

    • host_name: This is your database host name
    • database_name: Create a database name in your host (localhost or webhost) and put name here
    • username: Username of the database you created
    • password: Password of the database you created
